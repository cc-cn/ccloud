复现Mask_RCNN
参考：https://github.com/matterport/Mask_RCNN


进入tf1.3，cuda8环境
git clone https://github.com/matterport/Mask_RCNN
cd Mask_RCNN


验证keras版本
python
import tensorflow
import keras
print(tensorflow.__version__) ##1.3
print(keras.__version__)  ##2.2.0


如果是首次安装keras：直接在tf1.3环境下pip install keras，这样就会安装一个基于tf1.3的keras

下面安装requirements.txt文件里尚未安装的包
pip install opencv-python
pip install imgaug


编译，大约需要3分钟
python3 setup.py install

使用jupyternotebook生成的网址无法打开，查了一下可能和代理有关，使用如下命令：
jupyter notebook --ip 192.168.99.200 --port 8899
然后在火狐里打开生成的网址，成功


下面运行demo
问题：ImportError: No module named 'pycocotools'
参考https://github.com/matterport/Mask_RCNN/issues/6 里pirahagvp的高赞答案，可能还要参考YerongLi的答案，修改以下编译使用的python版本为/home/chencheng/anaconda3/envs/tensorflow/bin/python3.6
这里很奇怪，使用which python命令查看使用的python版本的时候，结果就是上述python啊！！

接着出现了TypeError: softmax() got an unexpected keyword argument 'axis'，应该是tf版本太旧，因此使用tf1.8
上面步骤再操作一遍，需要重新编译
然后demo就可以跑通了，在测试自己的图片的时候发现一个bug，就是当图片宽小于高的时候，会把图片旋转90度，错误率也提高了。
