//其他企业邮箱以自自己己公司邮箱服务器器为准
set smtp-auth-user="chchnju" smtp-auth-password="******" smtp-auth=login



发邮件有三种方法

1.交互式邮件发送:优势是格式灵活

mail + ****@mail.com
填写主题
填写内容
ctrl + d 结束输入入
cc代表抄送
回⻋车完成发送

2.管道:优势是发送日志提醒
echo 你好老师,我是陈成. | mail -s 问候 chchnju520@gmail.com

3.文件:优势是可以方便保留发送的文件,自制模板.
mail -s 问候 chchnju520@gmail.com < 文本文件路径

带附件的邮件:
mail -s 主题 -a attachment_name chch@pku.edu.cn < content.txt


163邮箱给gmail邮箱发送成功,给pku邮箱发送失败,但163这边显示已经发出.可能是教育邮箱兼容性不好?后来发现是被放到垃圾邮箱里了...hahahaha

mail命令很强大,更多用法可以通过 man mail命令查看
