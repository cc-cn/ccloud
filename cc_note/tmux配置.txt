安装：sudo apt-get install tmux

输入tmux进入tmux模式  

tmux kill-session -t foo # 删除名称为 foo 的会话
或者在进入tmux的情况下，ctrl b ：kill-session -t foo # 删除名称为 foo 的会话

竖直分割：Ctrl-b松开再按下% 
水平分割：Ctrl-b， "
窗格间移动光标：Ctrl-b，方向键
关闭窗格：Ctrl-b，x y
窗格全屏：ctrl-b z
创建一个窗口:Ctrl-b c
重命名窗口:Ctrl-b ,
切换窗口:Ctrl-b，窗口所对应的数字(正下方有显示)

复制粘贴：鼠标选中区域，松开鼠标，按ctrl-b-]可讲刚刚选中的区域复制到光标处。


创建新的会话：tmux new -s <name-of-my-session>  （会话是窗口集合，窗口是窗格集合。）


tmux a 命令，这样可以接入第一个可用的会话：
Ctrl-b d 断开已有的会话
Ctrl-b s 在会话间切换

ctrl-b 上下翻页键可实现翻页，esc退出翻页模式。

整行复制可按住ctrl在选择文本。。还是不完美

心得：通过terminal标签进入不同服务器，然后在各自的服务器中开启tmux。

修改默认的ctrl+b：tmux set -g prefix C-a （ctrl+a）
